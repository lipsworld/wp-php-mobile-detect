Wordpress wrapper for the Mobile Detect lightweight PHP class for detecting mobile devices.

== Description ==

Wordpress wrapper for the Mobile Detect lightweight PHP class for detecting mobile devices. Add simple mobile detection to your theme.

= Usage =

`<?php
if (is_mobile()) {
	// do mobile specific action
}
?>`

== Installation ==

1. Upload the `wp-php-mobile-detect` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
